## Development

### Running Tests

This project includes tests to help catch regressions when updating dependencies:

```bash
# Run tests once
npm test

# Run tests in watch mode
npm run test:watch

# Run tests with UI
npm run test:ui
```

### Building

```bash
npm run build
```

### Development Server

```bash
npm start
```
